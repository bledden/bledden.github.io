# bledden.github.io

Writing about patterns from ML experiments - the kind of thing that only becomes visible after running 250+ evaluation runs across multiple projects.

Topics: model orchestration, training dynamics, distillation, multi-agent systems.

## Local Development

```sh
npm install
npm run dev
```

Site runs at `localhost:4321`.

## Deploy

Pushes to `main` trigger automatic deployment via GitHub Actions.

## Built With

- [Astro](https://astro.build) - Static site generator
- [GitHub Pages](https://pages.github.com) - Hosting
